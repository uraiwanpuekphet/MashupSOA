# MashUp Update Movies and TV-Series
Mash Up จากการใช้ TMDB API เพื่อ update รายการหนังที่กำลังฉายและหนังที่กำลังเข้าฉาย สามารถ search ดูรายละเอียดของหนังแต่ละเรื่องได้
สามารถเลือกจากรายการได้ คือ
- Top Rated Movies
- Popular Movies
- Now Playing Movies
- Upcoming Movies
- TV SERIES
